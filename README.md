# Mandala Unfolding 🌀

Click-reactive mandala generator that creates beautiful, flowy, organic patterns with each click.

## Features

- **Click to Unfold**: Each click adds a new layer/fold to the mandala
- **Big, Flowy Patterns**: Large, organic petal shapes with smooth curves
- **Warm Color Palette**: Reds, oranges, golds, and purples
- **Radial Symmetry**: Perfect mandala structure
- **Golden Details**: Golden outlines and decorative dots
- **Animated**: Subtle rotation and flowy movement

## How to Use

1. Open `index.html` in a browser
2. Click anywhere on the canvas to create a mandala
3. Each click adds a new fold/layer
4. Click "Clear" to start over
5. Click in different locations to create multiple mandalas

## Controls

- **Click**: Add a fold (each click adds one layer)
- **Clear Button**: Reset and start fresh

That's it! Simple and beautiful.

## Files

- `index.html` - Main HTML
- `main.js` - Application logic
- `mandalaGenerator.js` - Mandala pattern generator
- `styles.css` - Styling

Enjoy creating beautiful mandalas! 🎨✨
